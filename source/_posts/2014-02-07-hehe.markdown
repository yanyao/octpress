在Mac下，zsh的环境下，LC_ALL和LANG这两个环境变量是不存在的，所以导致octopress在运行

``` plain
rake generate
```
时报错,解决的办法是在~/.zshrc中加入
``` bash
export LC_ALL=en_US.UTF-8
export LANG=en_US.UTF-8 
```
